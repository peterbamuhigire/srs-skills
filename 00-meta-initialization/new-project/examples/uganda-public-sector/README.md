# NIRA Citizen Registry

**Domain:** uganda

**Methodology:** Waterfall (change in `_context/methodology.md` if your project uses Agile or Hybrid).

## What this example demonstrates

A minimal but kernel-passing project workspace for the uganda domain. Every phase directory contains the smallest file set that satisfies the nine registered gates.

## How to use as a starter

```bash
cp -r 00-meta-initialization/new-project/examples/uganda-public-sector projects/<YourProject>
python -m engine validate projects/<YourProject>
```

Then customise `_context/vision.md`, `_context/stakeholders.md`, and `_context/features.md` with real project content.
