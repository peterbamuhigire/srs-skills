# Project Vision

**Project:** Kulima
**Client Contact:** Peter Bamuhigire
**Date:** 2026-04-04

## Problem Statement

Ugandan farmers — from smallholders with 2 acres to commercial estates with hundreds — lack an affordable, offline-capable, mobile-money-integrated farm management system that handles both crops and livestock in one app. Existing tools are either too expensive (Farmbrite at $49-179/month), too limited (Shambapro with data deletion on inactivity), or too narrow (Jaguza for IoT only, FarmTrace for traceability only). No competitor combines crop management, livestock management, IoT sensor integration, live CCTV surveillance, GPS animal tracking, supply chain traceability, financial records, and AI advisory in a single platform designed for African farming realities.

## Vision

Kulima is a multi-tenant SaaS Farm Management Information System (FMIS) providing web and mobile apps (Android + iOS) for individual farmers, farm businesses, cooperatives, and investors to manage farms, crops, livestock, finances, workers, inventory, IoT devices, CCTV surveillance, GPS tracking, supply chain traceability, and an AI-powered farm advisor — built Uganda-first, offline-capable, mobile-money integrated, and globally extensible.

**Tagline:** *from the farm, for the farm*

## Methodology

Hybrid (Water-Scrum-Fall) — formal requirements and design documentation up front, agile sprints for implementation, flexible approval gates at each phase. This is consistent with the methodology used for Academia Pro and Medic8.

**Hybrid detection note:** The project exhibits Water-Scrum-Fall characteristics — a detailed 2,037-line product specification exists before development begins, a 4-phase build sequence with formal revenue gates is defined, and an IEEE-compliant SRS process is being executed. Approval gates are observed at each phase but without a rigid formal sign-off ceremony.

## Goals

1. Replace paper-based farm record-keeping for Ugandan smallholder farmers with a digital system that works offline on low-spec Android phones
2. Provide commercial farms and cooperatives with enterprise-grade financial reporting, staff management, and supply chain traceability (EUDR compliance)
3. Integrate IoT sensor data (Jaguza ear tags), live CCTV surveillance, and GPS animal tracking into a single farm dashboard
4. Enable AI-powered pest/disease identification and personalised farm advisory using Claude API
5. Achieve 100 paying farmers within Phase 1, scaling to UGX 150M/month MRR by Phase 4

## Target Markets (launch order)

Uganda → Kenya, Tanzania, Rwanda → Nigeria, Ghana, Zambia → Global

## Languages at Launch

English, French, Portuguese, Luganda, Swahili, Kinyarwanda

## Stakeholders

| Stakeholder | Primary Need |
|---|---|
| Individual Farmer (smallholder) | Simple digital records, offline-capable, affordable, local language |
| Farm Business / Company | Multi-user access, formal accounting, investor/bank reports |
| Cooperative / NGO / Programme | Member registration, bulk input distribution, collective sales, payment distribution |
| Farm Director / Investor | Consolidated multi-farm view, approval workflows, performance monitoring |
| Buyer / Off-taker | Verifiable traceability data via QR code, EUDR compliance proof |
| Jaguza (IoT partner) | API integration, expanded market for their hardware |
| Banks / SACCOs / MFIs | Standardised farm financial data for loan applications |
| MAAIF / Government | Compliance reporting, extension service integration |

## Success Criteria

1. Phase 1 MVP launched with 100 paying farmers within 6 months of development start
2. Offline-first mobile app functions at full capacity with zero internet
3. Complete crop lifecycle tracking (plan → plant → activity → harvest) operational
4. Livestock individual animal tracking and herd management operational
5. Financial records generate bank-ready loan application reports
6. UGX 4M/month MRR achieved in Phase 1
