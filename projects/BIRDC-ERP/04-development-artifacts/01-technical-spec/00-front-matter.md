## Scope

This document defines the technical implementation standards for the BIRDC ERP system. It covers PHP backend standards, MySQL database conventions, JWT authentication, API response envelopes, CI/CD pipeline, security requirements, Android mobile app technical stack, and performance thresholds.

This specification is the authoritative technical reference for all BIRDC ERP development. All code, database objects, and API endpoints must conform to the standards herein. Deviations require written approval from the project consultant.
