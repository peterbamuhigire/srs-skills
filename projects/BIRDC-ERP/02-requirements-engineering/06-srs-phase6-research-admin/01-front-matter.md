## Document Conventions

- **Bold** identifiers denote requirement identifiers (e.g., **FR-RES-001**) and UI element names.
- *Italic* marks critical warnings and first introduction of defined terms.
- `Monospace` denotes file paths, terminal commands, API endpoints, database objects, and system identifiers.
- Every functional requirement follows the stimulus-response pattern: when [stimulus], the system shall [response], measured by [test oracle].
- `[CONTEXT-GAP: GAP-xxx]` flags an unresolved dependency that must be resolved before development of the tagged requirement begins.
