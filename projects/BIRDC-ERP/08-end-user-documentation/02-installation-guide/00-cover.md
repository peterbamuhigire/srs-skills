## About This Guide

This guide covers everything needed to set up the BIRDC ERP system for first-time use. It is intended for the IT Administrator and for any staff member installing one of the six Android apps on their phone.

**This guide covers:**

- Accessing the web application from a browser
- Installing each of the 6 Android apps
- First-time business configuration (accounts, users, roles)
- Connecting the ZKTeco biometric attendance device
- Understanding what works offline and what requires internet

**Who should read this guide:**

- IT Administrator (all sections)
- Finance Director (Sections 1 and 3)
- Any staff member receiving an Android app (Section 2)

If you encounter an error not described in this guide, contact the system support contact at techguypeter.com.
