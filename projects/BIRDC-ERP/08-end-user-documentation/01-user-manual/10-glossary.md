# Glossary of Terms

| Term | Plain-language meaning |
|---|---|
| Agent | A BIRDC field sales agent who sells Tooke products from a van in a defined area. BIRDC has 1,071 agents. |
| Agent Cash Balance | The total amount of cash an agent owes BIRDC. Calculated as: all sales collected minus all remittances that have been verified. |
| Agent Stock Balance | The quantity and value of Tooke products currently held by an agent for sale. Kept completely separate from warehouse stock. |
| BIRDC | Banana Industrial Research and Development Centre — the organisation that runs the Nyaruzinga factory. |
| Batch | A group of goods produced or received together, identified by a single batch number. |
| Certificate of Analysis (CoA) | A document from the QC lab certifying that a product batch meets all quality requirements. Required for export. |
| Cooperative | A group of smallholder farmers who deliver matooke together. |
| EFRIS | Uganda Revenue Authority's electronic system that records every sale and receipt. The ERP submits to EFRIS automatically. |
| FEFO | First Expiry First Out — the rule that the product expiring soonest is always sold or used first. |
| Fiscal Document Number (FDN) | The unique number URA gives to a confirmed sale. Printed on every receipt. |
| Float | Cash held in the till at the start of a POS shift. |
| Float Limit | The maximum value of stock an agent may hold at one time. |
| GL | General Ledger — the master record of all financial transactions. |
| GRN | Goods Receipt Note — the document confirming goods have arrived and been counted. |
| Hash Chain | A security feature that makes it mathematically detectable if any past financial record is changed. |
| Journal Entry | A double-entry accounting record — debits always equal credits. |
| LPO | Local Purchase Order — the standard procurement document used in Uganda. |
| LST | Local Service Tax — a small tax deducted from salaries by local government. |
| Mass Balance | The rule that every kilogramme of matooke that enters the factory must be accounted for as product, by-product, or waste. |
| Matooke | Fresh green bananas — the raw material BIRDC processes. |
| NCR | Non-Conformance Report — raised when a quality failure is found. |
| NIN | National Identification Number — Uganda's national ID number. |
| NSSF | National Social Security Fund — mandatory pension deductions. |
| OAG | Office of the Auditor General — Uganda's government auditor. |
| PAYE | Pay As You Earn — income tax deducted from salaries. |
| PIBID | Presidential Initiative on Banana Industrial Development — the government body that funds BIRDC. |
| POS | Point of Sale — any screen or device used to process an immediate cash or mobile money sale. |
| PPDA | Public Procurement and Disposal of Public Assets Authority — the body that regulates government procurement. |
| Remittance | Cash paid by an agent to BIRDC representing collected sales proceeds. |
| Session | The period from opening the cash register (with a float) to closing it at end of shift. |
| Sync | Uploading data from a phone app to the main server. |
| Tooke | BIRDC's brand name for its banana-based products (flour, chips, juice, and others). |
| Vote | A parliamentary budget category — money allocated by Parliament for a specific purpose. |
| WIP | Work in Progress — materials that are in the middle of being processed. |
| ZKTeco | The brand of fingerprint attendance machines installed at BIRDC. |
