# Section 5: Out of Scope

The following items are explicitly excluded from the BIRDC ERP project as currently specified. Items marked as "deferred" may be considered in a future engagement; items marked as "excluded" are outside the project boundary entirely.

## 5.1 iOS Mobile Applications (Deferred)

iOS versions of all 6 mobile applications are deferred. The current specification covers Android-only mobile development. BIRDC's field agent and factory worker population uses Android devices exclusively. iOS development would be scoped as a separate engagement if BIRDC's device landscape changes.

*iOS apps are out of scope for all 7 phases.*

## 5.2 Phase 8 and Beyond: New Module Categories

The following module categories are outside the BIRDC ERP scope for the current engagement. They represent potential future expansion but are not part of the current product requirements:

- Customer Relationship Management (CRM) with pipeline forecasting
- E-commerce or online ordering portal for Tooke products
- Third-party logistics (3PL) integration with external shipping carriers
- Business Intelligence (BI) platform or data warehouse beyond in-system reports
- Integration with any Enterprise Resource Planning system other than URA EFRIS and the specified mobile money APIs

## 5.3 Phase 7 Integration Items: In Scope

For the avoidance of doubt, the following integration items listed in Phase 7 are *in scope*:

- URA EFRIS system-to-system API (F-018)
- MTN MoMo Business API for agent remittances, farmer payments, and casual worker salary
- Airtel Money API (same use cases as MTN MoMo)
- ZKTeco biometric device integration for HR attendance
- Bluetooth ESC/POS thermal receipt printer integration (Sales Agent App, Farmer Delivery App)
- Bluetooth weighing scale integration (Farmer Delivery App)

## 5.4 Data Migration from Legacy Systems

Data migration from any existing legacy accounting or management system is out of scope for the initial build unless BIRDC provides complete, clean data exports and a funded migration sprint is agreed. [CONTEXT-GAP: GAP-014 — BIRDC's existing accounting software and data migration requirements are unconfirmed. Resolution required before go-live planning.]

## 5.5 Hardware Supply

Supply, procurement, or installation of server hardware, Android devices, thermal printers, barcode scanners, or biometric devices is outside the consultant's scope. BIRDC is responsible for hardware procurement. The consultant will provide hardware specification guidance.
