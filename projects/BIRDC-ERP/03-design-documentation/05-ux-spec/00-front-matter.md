## Scope

This document defines the user experience standards, screen inventory, key screen specifications, navigation architecture, mobile UX patterns, and DC-001 validation results for the BIRDC ERP system. It covers all 3 application panels, all 17 modules, and all 6 Android mobile apps.

This specification is binding on all front-end development work. Every screen produced must satisfy the design principles in Section 1 and pass the DC-001 validation checklist in Section 6.
