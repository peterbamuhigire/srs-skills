## Document Purpose

This High-Level Design document defines the architectural blueprint for the BIRDC Enterprise Resource Planning (ERP) system. It establishes the system context, three-panel application structure, deployment topology, application layer design, database architecture, dual-track inventory model, dual-mode accounting model, cryptographic audit mechanisms, security architecture, Android mobile architecture, external integration points, and the 17-module dependency map.

This document is the reference authority for all subsequent Low-Level Design (LLD), database schema design, API contract, and implementation decisions. Any architectural deviation from this document requires written approval from the Finance Director and BIRDC Director.
