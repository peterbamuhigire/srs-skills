# User Personas — Academia Pro

## Agnes Nakabugo (Accounts Bursar)

- **Age / Background:** 38, diploma in Business Administration, 10 years at St. Mary's Boarding School Kampala. Uses a Windows laptop at work; feature phone at home.
- **Goals:** Record all fee payments accurately; print receipts on demand; know which students owe fees before term begins; send SMS reminders without typing each parent's number.
- **Pain Points:** Current system (Excel + paper receipts) loses track of partial payments; she has to manually reconcile SchoolPay SMS notifications with her spreadsheet every day; double-entries happen and the owner blames her.
- **Tech Comfort:** Medium — comfortable with Excel, uses WhatsApp, struggles with unfamiliar software without a walkthrough.
- **Typical Workflow:** Opens Excel file each morning → checks phone for SchoolPay payment alerts → manually enters payments into spreadsheet → prints receipt from Word template → files paper copy → sends WhatsApp message to parents for balances.
