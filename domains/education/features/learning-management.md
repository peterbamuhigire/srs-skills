# Feature: Learning Management

## Description
Course content delivery, assignments, discussions, and student engagement
tracking — supporting both synchronous and asynchronous learning with
third-party tool integration via LTI 1.3.

## Standard Capabilities
- Course creation and curriculum structure (modules, units, lessons)
- Content authoring and upload (documents, video, SCORM packages, H5P)
- Assignment creation with submission workflow and plagiarism detection hooks
- Discussion boards and threaded communication
- Live session scheduling and video conferencing integration
- LTI 1.3 launch for third-party educational tools
- Course enrollment management and access control by enrollment status
- Student progress tracking and completion status
- Notification and announcement delivery (email, in-app)
- Accessibility-first content rendering (captions, transcripts, alt text)

## Regulatory Hooks
- WCAG 2.1 AA: all course content and UI must meet accessibility standards
- FERPA: course participation data (discussion posts, submission records) are education records
- COPPA: under-13 accounts must not have access to public discussion features
- CIPA: content filtering applies if the institution receives E-Rate funding

## Linked NFRs
- EDU-NFR-001 (Student Record Confidentiality — participation data is an education record)
- EDU-NFR-002 (Accessibility Compliance — all course content interfaces)
- EDU-NFR-005 (System Availability During Exam Periods)
