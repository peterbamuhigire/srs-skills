# Feature: Weather and Advisory

## Description
Farm-specific weather forecasting and climate-smart advisory services. Delivers real-time and seasonal weather data tied to each farm's GPS location, weather-triggered alerts via SMS and push notification, historical weather analysis, and actionable advisory for irrigation scheduling, planting windows, and frost risk management for highland farms.

## Standard Capabilities
- Farm-specific weather forecast using Open-Meteo API tied to GPS coordinates
- Real-time weather display (temperature, humidity, rainfall, wind) per farm location
- Weather alerts via SMS and push notification (heavy rain, drought, hailstorm)
- Seasonal forecast integration with Uganda Meteorological Authority data
- Historical weather data with trend analysis per farm
- Climate-smart agriculture advisory (planting timing, spray windows, harvest readiness)
- Frost alert system for highland farms (above 1500 m elevation)
- Irrigation scheduling recommendations based on rainfall forecast and soil moisture
- Weather impact correlation with crop yield and livestock production data
- Rainfall accumulation tracking per season

## Regulatory Hooks
- No specific regulatory requirements; advisory content is informational only
