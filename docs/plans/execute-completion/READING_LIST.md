## Workflow

1. Run the phase against current Claude capability — produces a first-draft skill.
2. Review draft against the free resources for that phase — fills obvious gaps.
3. Acquire Tier-1 books for that cluster before second-pass strengthening.
4. Re-run the skill enhancement pass after reading, citing book-specific
   frameworks, metrics, and patterns directly in the skill body.
5. Add Tier-2 books for depth on the third pass or when consulting demand
   warrants deeper coverage.
