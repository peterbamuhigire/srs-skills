# Deterministic Gate: Phase 04

## Scope

Phase 04 validates development-facing artefacts.

## Standard Anchor

- ISO/IEC/IEEE 12207:2017 clause 6.4.5

## Enforced Checks

- `phase04.coding_standards_referenced`
- `phase04.env_setup_reproducible`
- `phase04.tech_spec_links_to_fr`
- `phase04.contrib_guide_present`

## Intent

- Require coding standards under the development phase
- Require reproducible environment setup with prerequisites, installation, and verification
- Require FR-linked technical specifications
- Require a contribution guide

## Pass Condition

Implementation teams can reproduce the environment, follow explicit coding rules, and trace technical specifications back to requirements.
