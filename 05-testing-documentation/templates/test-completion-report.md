# Test Completion Report Template

Based on BS ISO/IEC/IEEE 29119-3 Section 7.6.

- **Report ID:** TCR-{YYYY}{NNN}
- **Project / Release:** [ProjectName / Release Name]
- **Date/Time:** [timestamp]
- **Prepared By:** [role/name]
- **Test Levels Covered:** Unit, Integration, System, UAT (per strategy)
- **Execution Summary:** Total test cases executed, pass/fail count, blocked.
- **Incident Overview:** Reference Incident IDs opened during execution.
- **Coverage Summary:** Requirement coverage percentage per traceability matrix.
- **Metrics:** Pass rate, defect density, mean time to resolution, automation coverage.
- **Exit Criteria:** List of entry/exit criteria per level and whether they were satisfied.
- **Deviations & Waivers:** Document `[WAIVER: reason]` tags where a clause/criterion was deferred.
- **Recommendation:** Go/Conditional Go/No-Go plus rationale referencing exit criteria.

Use this template as the final closing artifact for Phase 05 testing per 29119-3.
